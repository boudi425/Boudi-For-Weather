# Boudi-For-Weather

A comprehensive weather application designed to provide real-time weather information and forecasts.

## About

Boudi-For-Weather is a modern weather application built to deliver accurate and up-to-date weather data to users worldwide.

## Features

- **Real-time Weather Data**: Current temperature, humidity, wind speed, and atmospheric pressure
- **Weather Forecasts**: Extended forecasts for the upcoming days
- **Location Search**: Find weather information for any location globally
- **Weather Alerts**: Get notified about severe weather conditions
- **User-friendly Interface**: Clean and intuitive design for easy navigation
- **Responsive Design**: Works seamlessly across all devices

## Tech Stack

| Category        | Technology     |
| --------------- | -------------- |
| **Framework**   | React 19       |
| **Language**    | TypeScript     |
| **Build tool**  | Vite 7         |
| **Styling**     | Tailwind CSS 4 |
| **Weather API** | Weather API    |

## Installation

```bash
git clone https://github.com/Boudi425/Boudi-For-Weather.git
cd Boudi-For-Weather
npm install
npm start
```
